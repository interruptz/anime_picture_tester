Anime picture tester
====================

This program tests if a picture is a real photo or from anime based on neural network techniques.

in our given data set, it showed 95%+ accuarcy.


    $ pip install numpy neurolab PIL
    $ python main.py
    File path:test1.jpg
    [ -4.85186913e-09]
    Real picture
    $ python main.py
    File path:test2.jpg
    [ 1.]
    Anime picture
